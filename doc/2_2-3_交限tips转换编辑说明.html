获取进入线

业务说明：

获取退出线

获取限制信息

业务说明：

1、程序根据交限箭头对应的角度计算挂接在同一点或同一路口的退出线，交限箭头的对应角度范围如下：

禁直： (angle > 157.5 && angle <= 202.5)；

禁左：(angle > 247.5 && angle <= 292.5)；

禁调：(angle > 337.5 && angle <= 360.0)；(angle >= 0 && angle <= 22.5)；

禁右：(angle > 67.5 && angle < 112.5)；

2、若同一箭头方向在角度范围内寻找到多条与进入link挂接同一路口的退出link，根据以下情况进行判断；

①：tips未记录退出线信息，程序选择退出线与进入线夹角中与南北，东西方向夹角最小的一条作为退出线；

②：tips记录退出线信息，根据tips记录的退出线信息生成退出线

3、若同一箭头方向在角度范围内未找到进入线，则高亮该箭头方向，并提示"未计算出退出link，请人工指定退出线"；

4、限制信息编辑编辑栏选中某个箭头，该箭头的退出线并高亮体现在图面上，光标回到地图界面，默认可修改当前箭头对应的退出线(已选中高亮LINK即代表删除，点击未其他其他link即代表新增)：

5、退出线选定后，程序根据进入线和退出线自动判断关系类型，如果进入线和退出线直接挂接或挂接在同一路口上且均不为路口内道路，则关系类型为“路口关系”，其他情况为“线线关系”；

6、如果是路口关系，系统提示“路口挂接”；

7、如果是线线关系，系统提示“线线连接”；

8、人为指定该箭头的退出线后，程序自动计算经过线，系统经过最短路径计算选择经过线；选定经过线后，将经过线红色高亮显示；按p键可进入修改该箭头的经过线（左键点击不是经过线的link即代表增加，点击是经过线的LINK即代表删除），按c键程序重新计算该箭头的经过线

获取进入点

业务说明:

1、如果进入线是单方向道路，系统则自动选取link通行方向端点，作为进入点；

2、如果进入线是双方向道路，根据显示坐标和进入线自动计算通行方向，将该方向的LINK端点作为进入点，方向原则如下；

① 显示坐标到进入线画垂线；

② 显示坐标到垂足方向的顺时针（大陆）/逆时针（港澳）90°方向作为通行方向，；

 

业务说明:

1、系统根据tips的交限信息及退出线信息生成交限数据的限制信息，详细原则如下：

①：如果tips的箭头只有一个方向，则自动生成该箭头方向；

②：如果tips的箭头包含多个方向，则生成多个方向的单独箭头；

③；如果一个箭头包含多个退出线，则根据退出线个数生成多个箭头。

2、生成的限制信息高亮在map地图的右侧限制信息编辑栏中

3、根据箭头类型自动维护交限标志

4、根据箭头是否存在时间信息维护限制类型，如果箭头存在对应的时间信息，则为时间段禁止，否则为禁止驶入；

实时控制：

1、PERMIT_CHECK_IS_INTEGRITY_RDRESTRICTION：交限信息不完整，交限信息应该是完整的线点多线关系

2、GLM01017：“轮渡”和“人渡”的link不能作为交限（包括路口和线线结构里的所有交限）的进入线、退出线和经过线；

3、GLM26017：如果交限、语音引导、顺行进入线和退出线挂接在同一点上，而且这个点未登记路口（不属于任何路口），则不允许制作和修改；

4、RELATING_CHECK_NOSAME_LINE_LINE_RELATION：相同的进入线、进入点不能创建两组（车信、普通交限、顺行、分叉口提示、分歧、语音引导、收费站、大门、自然语音引导）；

5、重新编号：经过线没有闭合，请完成经过线设置

6、GLM08033：路口交限的进入线，退出线不能是交叉口内link

7、GLM01028_4：10级路/步行街/人渡不能是交限的进入线，退出线，经过线。

8、GLM08004_1：公交车专用道不能作为交限（包括路口和线线结构里的所有交限）的进入线或者退出线（也查线线经过线，但不查线线经过线为交叉口内link的情况）

9、GLM08004_2：步行街不能作为交限（包括路口和线线结构里的所有交限）的进入线或者退出线（也查线线经过线，但不查线线经过线为交叉口内link的情况）

10、GLM08004_5：出租车专用道（加了永久穿行限制，并且车辆类型同时且只制作了允许出租车、急救车和行人的link）不能作为交限（包括路口和线线结构里的所有交限）的进入线或者退出线（也查线线经过线，但不查线线经过线为交叉口内link的情况）

11、GLM08004_6：卡车专用道（加了永久穿行限制，并且车辆类型限制同时且只制作了允许配送卡车、运输卡车、急救车和行人的link）不能作为交限（包括路口和线线结构里的所有交限）的进入线或者退出线（也查线线经过线，但不查线线经过线为交叉口内link的情况）

12、GLM08004_7：进不去出不来的link（加了永久穿行限制，并且车辆类型限制同时且只制作了允许步行者、急救车的link）不能作为交限（包括路口和线线结构里的所有交限）的进入线或者退出线（也查线线经过线，但不查线线经过线为交叉口内link的情况）

13、GLM08031:路口交限里不允许有经过线信息

14、GLM08049：路口交限的进入线到退出线无通路可行，即报LOG

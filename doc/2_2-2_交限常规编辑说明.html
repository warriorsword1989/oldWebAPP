操作说明：

1、点击新增关系按钮

2、选择类型为“交限”，界面自动切换为“路口处关系”场景

3、map界面提示用户选择“进入线”，用户选择rd_link作为进入线；

4、map界面以绿色加粗高亮显示该Link，同时提示用户选择“进入点”；

5、用户选择进入线的某一个端点作为“进入点”；

6、map界面以绿色高亮显示进入线并加箭头显示进入点方向，以红色高亮显示进入点，同时map图面右上角弹出“限制信息”输入窗口；

7、点击选择对应的限制箭头，程序自动计算该箭头的退出link并高亮体现在图面上，同时该箭头在进入道路右侧显示，退出线上也显示该箭头；

8、点击link可修改该箭头对应的退出线，若程序提示未计算出对应的退出线，则人工指定该箭头的退出线

8、重复步骤7-8，输入其他方向限制箭头，并验证修改其退出线

9、空格键完成交限创建；

10、界面高亮交限进入，经过，退出线；同时属性栏面板展示属性信息，并提示用户进行属性栏编辑；

取消动作：

1、保存前，点击其他工具按钮，结束当前操作，切换至其他动作；

2、保存前，按键盘的“Esc”，回退至上一动作；

3、点击属性栏下方取消按钮，结束当前操作，不生成数据

业务要求：

1、默认开启线捕捉功能，系统根据鼠标移动捕捉最近的线；

2、如果进入线是单方向道路，系统则自动选取link通行方向端点，作为进入点；

3、限制信息选择：

      ①.左键点击下方菜单的限制箭头即为选中并新增箭头

      ②.点击已选中箭头，拖出交限信息框外，即为删除该箭头

删除箭头过程中自动删除对应的详细信息

4、程序根据交限箭头对应的角度计算挂接在同一点或同一路口的退出线，交限箭头的对应角度范围如下：

禁直： (angle > 157.5 && angle <= 202.5)；

禁左：(angle > 247.5 && angle <= 292.5)；

禁调：(angle > 337.5 && angle <= 360.0)；(angle >= 0 && angle <= 22.5)；

禁右：(angle > 67.5 && angle < 112.5)；

若某一箭头方向在角度范围内寻找到多条与进入link挂接同一路口的退出link，程序选择退出线与进入线夹角中与南北，东西方向夹角最小的一条作为退出线；

若某一个箭头方向在角度范围内未找到与进入link挂接同一路口的退出link，则程序报出提示:"未计算出退出link，请人工指定退出线"；

5、选中某个箭头，该箭头的退出线并高亮体现在图面上，光标回到地图界面，默认可修改当前箭头对应的退出线(已选中高亮LINK即代表删除，点击未其他其他link即代表新增)：

6、退出线选定后，程序根据进入线和退出线自动判断关系类型，如果进入线和退出线直接挂接或挂接在同一路口上且均不为路口内道路，则关系类型为“路口关系”，其他情况为“线线关系”；

7、如果是路口关系，系统提示“路口挂接”；

8、如果是线线关系，系统提示“线线连接”；

9、人为指定该箭头的退出线后，程序自动计算经过线，系统经过最短路径计算选择经过线；选定经过线后，将经过线红色高亮显示；按p键可进入修改该箭头的经过线（左键点击不是经过线的link即代表增加，点击是经过线的LINK即代表删除），按c键程序重新计算该箭头的经过线

1、PERMIT_CHECK_IS_INTEGRITY_RDRESTRICTION：交限信息不完整，交限信息应该是完整的线点多线关系

3、GLM26017：如果交限、语音引导、顺行进入线和退出线挂接在同一点上，而且这个点未登记路口（不属于任何路口），则不允许制作和修改；

13、GLM08031:路口交限里不允许有经过线信息

14、GLM08049：路口交限的进入线到退出线无通路可行，即报LOG
